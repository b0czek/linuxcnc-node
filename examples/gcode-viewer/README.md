# G-code Viewer Example

A 3D G-code visualizer built using Electron, Vite, Three.js, and the `@linuxcnc-node/gcode` parser.

## Prerequisites

- Linux system with LinuxCNC installed.
- Node.js 18 or later.

## Getting Started

1.  **Install dependencies**:

    ```bash
    npm install
    ```

2.  **Build the G-code native module**:
    Ensure you are in the project root and have initialized the `gcode` module.

3.  **Run the application**:
    ```bash
    npm run dev
    ```

    Interface EmcPose

    Position and orientation in the LinuxCNC coordinate system. Represents a pose with 9 degrees of freedom (3 linear + 3 rotational + 3 auxiliary). All values are in machine units. Axes descriptions are assumed, but they may vary by machine configuration.

    interface EmcPose {

        w: number;
        x: number;
        y: number;
        z: number;
    }
